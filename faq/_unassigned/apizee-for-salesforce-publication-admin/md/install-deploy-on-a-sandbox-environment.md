# install-deploy-on-a-sandbox-environment

1. Go to Apizee listing page on AppExchange: [https://appexchange.salesforce.com/appxListingDetail?listingId=d4c03675-1f40-4a1a-b121-b9704fe7d50c](https://appexchange.salesforce.com/appxListingDetail?listingId=d4c03675-1f40-4a1a-b121-b9704fe7d50c). ![](../../../.gitbook/assets/install-apizee-from-appexchange-2025-03-14.png)
2. Click **Try It Free** to install the app on a sandbox environment, then proceed with the app installation on your sandbox environement.
3. Click **Try in your sandbox.** ![](../../../.gitbook/assets/install-apizee-for-salesforce-app-from-appexchange-2025-03-11.png)
4. Complete your details and click **Continue to Installation** button.
5. Log in on the Salesforce sandbox environment you want to install the Apizee app on.
6. Click **Install for Admins Only**.
7. Click **Install.** ![Une image contenant texte, capture d’écran, Police, logo  Description générée automatiquement](../../../.gitbook/assets/img0005.png)

![A screenshot of a computer  Description automatically generated](../../../.gitbook/assets/img0006.png)
