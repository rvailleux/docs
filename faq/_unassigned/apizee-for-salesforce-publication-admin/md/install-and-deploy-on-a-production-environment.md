# install-and-deploy-on-a-production-environment

1. Go to Apizee listing page on AppExchange: [https://appexchange.salesforce.com/appxListingDetail?listingId=a94d3d81-9904-42c9-8145-37d26c926fe7](https://appexchange.salesforce.com/appxListingDetail?listingId=d4c03675-1f40-4a1a-b121-b9704fe7d50c)![](../../../.gitbook/assets/install-and-deploy-on-a-sandbox-environment-2025-08-04.png)
2. Click **Get It Now** to install the app on a production environment.
3. Select the appropriate Salesforce account to install the Apizee app on your production environment.
4. Click **Install In Production.**![](../../../.gitbook/assets/install-and-deploy-on-a-production-environment-2025-08-04-1.png)
5. Check the details. ![](../../../.gitbook/assets/install-and-deploy-on-a-production-environment-2025-08-04-2.png)
6. Click **Confirm and Install.**

![A screenshot of a computer  Description automatically generated](../../../.gitbook/assets/img0008.png)

| ![](../../../.gitbook/assets/ok.png) | The Apizee app is now installed on your production instance. |
