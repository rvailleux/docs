# configure-the-quick-invitation

1. On the left-hand menu, choose the service for which you want to configure the **Quick invitation**.
2. On the top right, click the switch button **Quick invitation**.

![](../../../.gitbook/assets/portal-configure-quick-invitation.gif)

{% hint style="success" %}
The configuration is automatically saved. On the left-hand menu, you can see a clock ![](../../../.gitbook/assets/clock-picto.png)in front of the service you just configured.
{% endhint %}

{% hint style="info" %}
If you have several services in the left-hand menu, note that you can configure the **Quick invitation** for one video conference and one video assistance service only.- the conference services have this pictogram ![](../../../.gitbook/assets/conference-picto.png)- the video assistance services have this pictogram![](../../../.gitbook/assets/ticket-picto.png)
{% endhint %}
