# change-my-status

1. On the top right, click on your **Profile**.
2. Choose your status in the drop-down menu:

* Available
* Busy
* Unavailable

![](<../../../../.gitbook/assets/EN-portal-change-status (1).gif>)

{% hint style="info" %}
The directory and the communication features are deactivated when the status is **Unavailable**.
{% endhint %}

{% hint style="success" %}
The page reloads with the new selected status.
{% endhint %}
