# about-apizee-solutions

Apizee solutions are "Plug-in free". They are 100% Web-based to ensure communication experience on major Internet browsers and connected devices.

Video communication has never been that easy!

\| No installation needed | The Apizee solutions use the **ApiRTC** architecture which uses the **WebRTC** protocol. This protocol of communication is directly embedded in the latest versions of the Web browsers. This is why you do not need to install anything as long as you have a compatible Web browser. ![](../../.gitbook/assets/schema_apirtc.png) | | --- | --- | | Compatible devices | The Apizee solutions are compatible with:
