# with-an-email-confirmation

1. Enter the user email address.
2. Click **Save**.

![](../../../../.gitbook/assets/EN-portal-create-user-email-confirmation.png)

{% hint style="success" %}
The account is created.

An email is sent to the new user with a validation link and a login.
{% endhint %}
