# the-conferences-are-they-confidential

**Apizee** solutions use the **WebRTC** protocol.

WebRTC offers encrypted flows between each entity ensuring real-time security and confidentiality of the communications.

For this, WebRTC relies on the SRTP\*\*\*\*\*\*\* _**and DTLS**\*\*_\*\* protocols.

_\*_&#x53;RTP: Secure Real-time Transport Protocol

_\*\*_&#x44;TLS: Datagramme Transport Layer Security
