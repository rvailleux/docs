# remote-assist-share-file

{% hint style="info" %}
You are the organizer of the session and you want to help a participant to share a file.
{% endhint %}

1. On the right, click the **Actions** tab&#x20;
2. Click the **Guests** tab.

&#x20;3\. If you are more than 2 participants, choose the name of the participant in the drop-down menu. 4. Click **Share file**.

```
| ![](../../.gitbook/assets/ok.png) | An invitation is sent to the participant and displays on his screen as follow: |
| --- | --- |
```

**Guest screen**&#x20;

```
| ![](../../.gitbook/assets/ok.png) | When accepted, a file explorer opens on the participant screen. 
```

He needs to choose a file and click **Open**. Once downloaded, the files displays in the **Messages** tab. | | --- | --- |
