# from-csv-file

{% hint style="info" %}
You want to create several accounts at the same time.
{% endhint %}

1. Open a spreadsheet document, such as Excel.
2. Fill in the table with:
   * First-name in column A
   * Last-name in column B
   * Username in column C
   * Password in column D
   * Email in column E
   *   Nickname in column F

       \|  | Do no use spaces for the nicknames.  Example: ~~docteur du jardin~~ > **docteur\_du\_jardin** or **docteurdujardin** | | --- | --- |

&#x20;3\. **Save** the document in .csv format.

&#x20;4\. On the Apizee portal, **choose** the document you saved earlier. 5. Choose the delimiter: **Semicolon (;)** 6. Click **Import**.

{% hint style="success" %}
The accounts are created.
{% endhint %}
