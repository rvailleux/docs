# share-a-file

## Share files:

* [During the session](share-a-file.md)
* [After the session in the portal](share-a-file.md)

### During the session

1. On the right, clique the **Messages**tab&#x20;
2.  At the bottom, next to the message field, click the **paper clip** button&#x20;

    |   | The file explorer opens on your screen. |
    | - | --------------------------------------- |

&#x20;3\. Choose a file and click **Open**.

```
| ![](../../../.gitbook/assets/ok.png) | When the file is uploaded, it displays in the **Messages**. |
| --- | --- |
```

{% hint style="success" %}
The participants will be able to download the file by clicking If you are the organizer of the session, you can open a picture in whiteboard to annotate or draw on it by clicking&#x20;
{% endhint %}

##

### After the session in the portal

{% hint style="info" %}
You are the organizer of the session from which the files are from. Or, you are an administrator. You are logged in to your account.
{% endhint %}

1. In the left-hand menu, click the service you want.
2. On the right, click **List**. That will help you find faster the session you are looking for.
3. In the list, find the session you want and click&#x20;

![](../../../../.gitbook/assets/EN-portal-teleconsultation-edit-teleconsultation-page.png)

```
| ![](../../../.gitbook/assets/ok.png) | The page displays. |
| --- | --- |
```

4\. Under **Shared files**, choose the files and click **Share**.

```
| ![](../../../.gitbook/assets/ok.png) | A window displays with the contacts of the guests that were invited during the session. |
| --- | --- |
```

5\. Click **Send**.

```
| ![](../../../.gitbook/assets/ok.png) | The **Sharing history** indicates the number of files shared and the remaining time before the link expires. |
| --- | --- |
```

```
| ![](../../../.gitbook/assets/ok.png) | The persons to whom you shared the files receive a message with a new download link. |
| --- | --- |
```
