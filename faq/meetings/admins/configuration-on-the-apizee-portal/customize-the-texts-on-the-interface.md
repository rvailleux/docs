# customize-the-texts-on-the-interface

You can customize and adapt the text of the interface.

1. In the left-hand menu, click the service for which you want to customize the interface texts.
2. Click the **Terminology** tab.

&#x20;3\. Press **CTRL+F** to search a keyword or the text you want to change. 4. Customize the sentence. 5. Scroll down the page and clic **Save**.

{% hint style="success" %}
The text you changed is surrounded in green so you can recognize the customized sentences.
{% endhint %}

{% hint style="success" %}
Your changes display on the interface
{% endhint %}
