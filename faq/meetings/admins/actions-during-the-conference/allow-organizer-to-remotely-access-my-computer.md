# allow-organizer-to-remotely-access-my-computer

{% hint style="info" %}
Available only on computers with Windows 10 & 11. [More information](../about-apizee-solutions.md#Browser-remote-access) about compatibility.
{% endhint %}

{% hint style="info" %}
You need the organizer to help you and want him to remotely access your computer.
{% endhint %}

1. When the organizer sends you an invitation, click **Allow**.

&#x20;2\. Follow the steps displayed on your screen: 1. Click **Share the screen**.

&#x20;2\. Make sure you select the **entire screen**. Here is an example for Chrome Web browser:

&#x20;3\. Click **Download** to download the Apizee remote access software application.

&#x20;4\. Once it is completely downloaded, launch the application by double-clicking the file **.exe**. 5. Accept the safety warnings displayed by Windows. \* Click **More info** \* Click **Run anyway**<br>

```
        | ![](../../.gitbook/assets/ok.png) | The remote access session starts. 
```

The organizer sees your screen and can control your mouth and keyboard. | | --- | --- |

&#x20;3\. When you are done, click&#x20;

{% hint style="success" %}
The remote access stops and the agent cannot control your computer anymore.
{% endhint %}
