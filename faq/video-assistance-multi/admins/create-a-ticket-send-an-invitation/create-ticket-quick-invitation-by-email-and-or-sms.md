# create-ticket-quick-invitation-by-email-and-or-sms

1. In the menu bar, at the top right, click **Quick invitation**.&#x20;
2. In the drop-down menu, choose the video assistance service (flow) with which you want to send the invitation.
3. Enter the guest email, phone number or a name if you use the **Directory**.

&#x20;3\. Click **Send invitation**.

{% hint style="success" %}
The invitation is sent.

All the guests will receive a message with a link to join the video assistance.

The assistance page displays on the agent screen.
{% endhint %}
