# create-a-new-service

1. In the left-hand menu, under **Configuration**, click **Services management**.
2. Click **Add service**.
3. Choose the service type\* according to your need:

```
| Video assistance | For video assistance with ticket management system.
```

```
| ![](../../.gitbook/assets/Apizee-see-also-icon.png) | **See also** [our **Apizee for Video assistance** user guide](../video-assistance/agents/about-apizee-video-assistance.md) |
| --- | --- | |
| --- | --- |
| Meeting | For video conference, meeting or individual appointments.
```

```
| ![](../../.gitbook/assets/Apizee-see-also-icon.png) | **See also** [our **Apizee for Meetings** user guide](../meetings/users/about-apizee-meetings.md) |
| --- | --- | |
| Customer engagement | For chatbox and video calls embedded on your website.
```

```
| ![](../../.gitbook/assets/Apizee-see-also-icon.png) | **See also** [our Apizee for **Customer engagement** user guide](https://doc.apizee.com/smart/project-contact-customer-relationship/about-apizee-customer-engagement) |
| --- | --- | |
```

\* The type of services displayed on your interface depends of the product you purchased. 4. Enter the name of your service.

```
| ![Information](../../.gitbook/assets/info.png) | The service will display with this name in the portal menu. |
| --- | --- |
```

5\. Choose the type of assistance you want. 6. Click **Add**.

{% hint style="success" %}
The new service displays in the left-hand menu
{% endhint %}
