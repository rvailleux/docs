# create-a-ticket-send-an-invitation

There are several ways to create an assistance ticket and send an invitation to the requester.

* [Quick invitation by email and/or SMS](create-ticket-quick-invitation-by-email-and-or-sms.md)
* [Quick invitation by SMS](create-ticket-quick-invitation-by-sms.md)
* [Common invitation](create-ticket-common-invitation.md)

{% hint style="info" %}
These different ways may depend on the configuration of the product in your Apizee portal.
