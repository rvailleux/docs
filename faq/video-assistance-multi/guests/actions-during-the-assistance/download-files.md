# download-files

### During the session

1. You received a notification in the **Messages**tab, click on the tab.
2. Click the **arrow** next to the image to download the file ![](../../../.gitbook/assets/button-download-apizee-meet.png)

![](../../../.gitbook/assets/EN-conf-guest-download-file-in-messages-tab-click.png)

{% hint style="success" %}
The file is downloaded.
{% endhint %}

### After the session

{% hint style="info" %}
The session is over now. The end of session page opens with the shared files on the right.
{% endhint %}

1. Click the **arrow** next to the file to download it. **OR**
2. Click **Download all** to download all the files.

![](../../../.gitbook/assets/EN-conf-end-of-session-page-shared-files-click.png)

{% hint style="success" %}
The files are downloaded.
{% endhint %}

If you want to, you can download those same files on another device. Interested? Click **Show more**.

\[+] [Show More](https://github.com/rvailleux/docs/tree/master/faq/video-assistance-multi/guests/actions-during-the-assistance/javascript:void\(0\)/README.md) \[-] [Hide](https://github.com/rvailleux/docs/tree/master/faq/video-assistance-multi/guests/actions-during-the-assistance/javascript:void\(0\)/README.md)

1. At the bottom right, click **Download on another device**.

![](../../../.gitbook/assets/EN-conf-end-of-session-page-download-files-on-another-device-button.png) 2. Enter en email address and click **Confirm**.

![](../../../.gitbook/assets/EN-conf-end-of-session-page-download-files-on-another-device-enter-address.png)

```
| ![](../../.gitbook/assets/ok.png) | A new message is sent on your email address |
| --- | --- |
```

3\. From the other device, open the new message you have been sent on the email address given below. 4. Click the link in the message.

```
| ![](../../.gitbook/assets/ok.png) | The end of session page opens with the shared files on the right. |
| --- | --- |
```
