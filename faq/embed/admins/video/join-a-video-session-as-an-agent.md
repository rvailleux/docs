# How to join a video session as an agent

{% hint style="info" %}
You just sent an invitation to a guest to join the video assistance.
{% endhint %}

* After sending the invitation, you will be able to follow the guest throughout their journey in the Apizee application.
* At the end of the guest journey, a devices usage authorisation pop-in appears.
* Allow the Web browser to use the camera and microphone.

{% hint style="success" %}
The video session starts.
{% endhint %}
