# webhook-subscriptions

{% hint style="info" %}
You have the permission to manage the webhooks.
{% endhint %}

Webhook subscriptions let you automatically send notifications to an external tool when an event occurs in Apizee Embed.

You manage webhook subscriptions from the **Information** tab.

For each subscription, the list shows the name, the destination URL, the trigger events, the last update date, and the status.

Click a row to open and edit the subscription.
