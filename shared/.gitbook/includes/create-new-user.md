{% hint style="info" %}
- You are logged in to the Portal.- You have permission to create users.- You know the user name, first name, role, and email address.
{% endhint %}

1. Select the **Information** tab.
2. Click **See All** next to the number of users.
3. Click **Add** on the users page.
4. Enter the **Name**.
5. Enter the **First name**.
6. Choose a **Role** from the drop-down list.
7. Enter the **email address**.
8. Click **Create**.

| ![](../../.gitbook/assets/ok.png) | The user is created. <br><br> The new user receives an email with a link to create a password. |
    | --- | --- |
9. Click **SEE** in the confirmation banner to be redirected to the user information page.
