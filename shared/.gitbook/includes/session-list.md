# Session list

1. Select the **Session** tab.

{% hint style="success" %}
The list of the sessions displays.
{% endhint %}

The list is sorted by date and hour.
You can check the guests and the state of the session.

# Session details

1. Select the **Session** tab.
2. In the list, click the session you want.

{% hint style="success" %}
The session details display.
{% endhint %}
