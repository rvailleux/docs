# How to join a video session as a guest

{% hint style="info" %}
You received a text message.
{% endhint %}

1. In the message, tap the link.

    {% hint style="info" %}

    The guest app opens in their browser.

    {% endhint %}
2. Tick the boxes to Agree to the legal terms.
3. Click **Continue**.
4. Allow the Web browser to use the microphone and camera.
5. Check your settings (camera, microphone).
6. Click **Join session**.

{% hint style="success" %}
The video session page displays.
{% endhint %}
