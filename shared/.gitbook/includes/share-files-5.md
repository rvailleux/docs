#  Share files:

* [During the session](communicate-colleagues-share-files.md)
* [After the session in the portal](communicate-colleagues-share-files.md)

## During the session

1. On the right hand-side, click the **paper clip**. 

![](../../.gitbook/assets/Diag-portal-agent-share-file.png)
2. Choose the file you want to share.
3. Click **Open**. 

![](../../.gitbook/assets/EN-Diag-portal-agent-share-file-choose-click-open.png)

{% hint style="success" %}
The file charges then, it displays in the conversation window on the right.
{% endhint %}

 ![](../../.gitbook/assets/Diag-portal-agent-share-file-displays-in-messages.png)

{% hint style="warning" %}
Unlike the **agent**, all the files shared by the **requester** are available after the session:
-  in the portal on the ticket page under **Shared media**. -  in the ticket public page.   
If you are an **agent**, the files you share will not display. Only the pictures you opened in the whiteboard will display on the pages below.
{% endhint %}

##  After the session in the portal

{% hint style="info" %}
You are the organizer of the session from which the files are from. Or, you are an administrator.
 You are logged in to your account.
{% endhint %}

1. In the left-hand menu, click the service you want.
2. In the list, find the session you want and click ![](../../.gitbook/assets/button-edit.png) 
 
 ![](../../.gitbook/assets/EN-portal-service-assistance-edit-ticket.png) 

{% hint style="success" %}
The page displays.
{% endhint %}
3. Under **Shared files**, choose the files and click **Share**. 
 
 ![](../../.gitbook/assets/EN-Diag-portal-ticket-page-share-file-after-session.png) 

{% hint style="success" %}
A window opens.
{% endhint %}
4. Enter en email address.
5. Click **Send**. 

{% hint style="success" %}
The persons to whom you shared the files receive a message with a new download link.
{% endhint %}
 ![](../../.gitbook/assets/EN-conf-message-files-shared-again.png)
