1. Sélectionnez l'onglet **Informations**.
2. Dans la section **Abonnements webhook**, cliquez sur la ligne de l'abonnement que vous souhaitez modifier.

    {% hint style="success" %}

    Les détails de l'abonnement s'affichent.

    {% endhint %}
3. Modifiez les champs selon vos besoins :
    * Pour les champs texte (Nom, URL, en-têtes) : cliquez sur le champ, saisissez vos modifications, puis cliquez sur l'icône **confirmer** pour enregistrer ou sur l'icône **annuler** pour ignorer.
    * Pour la méthode : sélectionnez une valeur dans la liste déroulante.
    * Pour les événements déclencheurs : cliquez sur une étiquette pour la sélectionner ou la désélectionner.

Vous ne pouvez pas modifier le secret. Il est généré par le système et affiché à titre de référence uniquement.
