1. Sélectionnez l'onglet **Informations**.
2. Dans la section **Abonnements webhook**, cliquez sur la ligne de l'abonnement que vous souhaitez supprimer.

    {% hint style="success" %}

    Les détails de l'abonnement s'affichent.

    {% endhint %}
3. Cliquez sur **Supprimer**.
4. Dans la boîte de dialogue de confirmation, cliquez sur **Supprimer** pour confirmer.

![](../../../.gitbook/assets/portal-webhook_subscription-delete.png)

{% hint style="success" %}
L'abonnement est définitivement supprimé et n'apparaît plus dans la liste.
{% endhint %}
