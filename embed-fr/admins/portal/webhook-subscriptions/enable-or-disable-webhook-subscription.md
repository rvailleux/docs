1. Sélectionnez l'onglet **Informations**.
2. Dans la section **Abonnements webhook**, trouvez l'abonnement.
3. Dans la colonne **Statut**, cliquez sur le bouton bascule pour activer ou désactiver l'abonnement.

![](../../../.gitbook/assets/portal-webhook_subscription-toggle.png)

{% hint style="success" %}
Le statut est mis à jour immédiatement. Un abonnement désactivé cesse d'envoyer des notifications.
{% endhint %}

Vous pouvez également modifier le statut depuis la page de détail de l'abonnement.
