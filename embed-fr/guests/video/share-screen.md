# Comment partager votre écran

{% hint style="info" %}
Vous êtes maintenant dans une session vidéo.
{% endhint %}

1. Dans la barre d'outils, cliquez sur ![](../../.gitbook/assets/screenshare.png)si vous souhaitez partager votre propre écran.
2. Choisissez l'écran que vous souhaitez partager.
3. Si vous souhaitez partager une vidéo et que vous voulez que les participants entendent le son de la vidéo, cochez la case **Partager l'audio**.
Cette option peut dépendre de votre navigateur Web.
4. Cliquez sur **Partager**.
![](../../.gitbook/assets/EN-Diag-share-screen-web-browser.png)
5. Pour arrêter le partage d'écran, cliquez à nouveau sur le bouton ![](../../.gitbook/assets/screenshare.png).
