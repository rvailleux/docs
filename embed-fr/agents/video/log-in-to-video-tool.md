# Authentification dans l'outil vidéo Apizee Embed

{% hint style="info" %}
*Vous devez disposer d'un compte Apizee pour vous authentifier.*

{% endhint %}

1. Cliquez sur **Se connecter**.
2. Une fenêtre contextuelle s'ouvre.
    1. Saisissez votre adresse email et votre password.
    2. Cochez **Se souvenir de moi** si vous souhaitez prolonger votre session d'authentification.
    3. Cliquez sur **Sign In**.
3. Si les informations de connexion sont correctes, la fenêtre contextuelle d'authentification se ferme automatiquement.

{% hint style="success" %}
*Vous êtes maintenant connecté. Vous pouvez utiliser Apizee Embed.*
{% endhint %}
