# Comment activer la lampe de poche du participant

{% hint style="info" %}
Vous êtes maintenant dans une session vidéo.
Le smartphone du participant utilise la caméra appropriée pour activer cette fonctionnalité.
{% endhint %}

* Dans la barre d'outils, cliquez sur ![](../../.gitbook/assets/turn-on-guest-flashlight-2025-03-05-1.png)pour activer ou désactiver la lampe de poche de l'appareil distant.

{% hint style="success" %}
Vous pouvez voir la lampe de poche s'allumer et s'éteindre directement sur le flux vidéo du participant.
{% endhint %}
