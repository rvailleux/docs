# Liste des sessions

1. Sélectionnez l'onglet **Session**.

{% hint style="success" %}
La liste des sessions s'affiche.
{% endhint %}

La liste est triée par date et heure.
Vous pouvez vérifier les invités et l'état de la session.

# Détails d'une session

1. Sélectionnez l'onglet **Session**.
2. Dans la liste, cliquez sur la session souhaitée.

{% hint style="success" %}
Les détails de la session s'affichent.
{% endhint %}
