# Comment configurer mes paramètres

{% hint style="info" %}
Vous êtes maintenant dans une session vidéo.
En tant qu'invité : vous vérifiez vos paramètres avant d'entrer dans la session vidéo.
{% endhint %}

1. Pendant une session vidéo, dans la barre d'outils, cliquez sur ![](../../.gitbook/assets/configure-my-video-session-settings-2025-03-05.png)pour accéder au menu de configuration.
2. Ensuite, pour chaque périphérique (caméra et microphone), vous pouvez :
    * Activer ou désactiver le périphérique.
    * Sélectionner le périphérique que vous souhaitez utiliser.

{% hint style="success" %}
Vous disposez de votre flux vidéo pour voir les modifications s'appliquer directement.
{% endhint %}
