# Comment désactiver et activer mon microphone et ma caméra pendant une session

{% hint style="info" %}
Vous êtes maintenant dans une session vidéo.
En tant qu'invité : vous vérifiez vos paramètres avant d'entrer dans la session vidéo.
{% endhint %}

### Activer/Désactiver la caméra

* Dans la barre d'outils, cliquez sur ![](../../.gitbook/assets/cameraOn.png)pour désactiver et sur ![](../../.gitbook/assets/CameraOff.png) pour activer la caméra.

### Activer/Désactiver le microphone

* Dans la barre d'outils, cliquez sur ![](../../.gitbook/assets/microphoneOn.png)pour désactiver et sur ![](../../.gitbook/assets/microphoneOff.png)pour activer le microphone.

{% hint style="success" %}
Vous disposez de votre flux vidéo pour voir les modifications s'appliquer directement.
{% endhint %}
