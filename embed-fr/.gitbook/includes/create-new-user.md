{% hint style="info" %}
- Vous êtes connecté au Portail.- Vous avez la permission de créer des utilisateurs.- Vous connaissez le nom, le prénom, le rôle et l'adresse email de l'utilisateur.
{% endhint %}

1. Sélectionnez l'onglet **Informations**.
2. Cliquez sur **Voir tout** à côté du nombre d'utilisateurs.
3. Cliquez sur **Ajouter** sur la page des utilisateurs.
4. Saisissez le **Nom**.
5. Saisissez le **Prénom**.
6. Choisissez un **Rôle** dans la liste déroulante.
7. Saisissez l'**adresse email**.
8. Cliquez sur **Créer**.

| ![](../../.gitbook/assets/ok.png) | L'utilisateur est créé. <br><br> Le nouvel utilisateur reçoit un email avec un lien pour créer un mot de passe. |
    | --- | --- |
9. Cliquez sur **VOIR** dans la bannière de confirmation pour être redirigé vers la page d'informations de l'utilisateur.
