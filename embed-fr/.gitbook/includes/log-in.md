# Se connecter

{% hint style="info" %}
Vous disposez de l'URL et vous avez cliqué dessus.
La page de connexion s'affiche.
{% endhint %}

1. Saisissez votre adresse **email**
2. Saisissez votre **mot de passe** et
3. Cliquez sur **Se connecter**.

Ou connectez-vous avec un compte externe, si cette option est disponible.

1. Sous **Ou se connecter avec**, cliquez sur le bouton correspondant au compte externe avec lequel vous souhaitez vous connecter.

{% hint style="success" %}
La page se charge et vous êtes connecté.
{% endhint %}

# Se déconnecter

{% hint style="info" %}
Vous êtes connecté.
{% endhint %}

1. En haut à droite, cliquez sur **Se déconnecter**.

{% hint style="success" %}
Vous êtes redirigé vers la page de connexion.
{% endhint %}
