# Comment rejoindre une session vidéo en tant qu'agent

{% hint style="info" %}
Vous venez d'envoyer une invitation à un invité pour rejoindre l'assistance vidéo.
{% endhint %}

* Après l'envoi de l'invitation, vous pourrez suivre l'invité tout au long de son parcours dans l'application Apizee.
* À la fin du parcours de l'invité, une fenêtre contextuelle d'autorisation d'utilisation des périphériques apparaît.
* Autorisez le navigateur Web à utiliser la caméra et le microphone.

{% hint style="success" %}
La session vidéo démarre.
{% endhint %}
