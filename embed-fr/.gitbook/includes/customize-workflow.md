1. Sélectionnez l'onglet **Flux**.

| ![](../../.gitbook/assets/ok.png) | Un fichier JSON s'affiche dans un éditeur. <br> Ce fichier contient toute la configuration liée au workflow de l'outil vidéo. |
    | --- | --- |
2. Cliquez dans l'éditeur pour modifier les éléments suivants :
    1. Afficher ou masquer le **pointeur à l'écran**.
L'utilisateur peut voir et utiliser le pointeur à l'écran.
    2. Afficher ou masquer le **partage d'écran**.
 L'utilisateur peut partager son écran avec les autres participants.
    3. Afficher ou masquer la **capture photo à distance**.
 L'utilisateur peut prendre une photo à distance.
    4. Afficher ou masquer le **bouton de changement de caméra**.
 L'utilisateur peut changer de caméra pendant l'appel.
    5. Afficher ou masquer le **contrôle de la torche à distance**.
 L'utilisateur peut allumer ou éteindre la lumière du téléphone.
    6. Afficher ou masquer le **menu d'aide**.
 L'utilisateur peut accéder à l'aide.
    7. Choisir la **caméra par défaut** pour l'invité principal.

        * Caméra avant.
        * Caméra arrière.
    8. Définir le **nombre maximum d'invités** autorisés.
    9. Activer ou désactiver la **fonction caméra**.

        * Caméra activée : l'image peut être affichée.
        * Caméra désactivée : aucune image n'est affichée.
    10. Arrêter ou autoriser l'**image de la caméra**.

        * Caméra arrêtée : l'image n'est pas envoyée.
        * Caméra autorisée : l'image est envoyée.
    11. Activer ou désactiver la **fonction microphone**.

        * Microphone activé : le son peut être envoyé.
        * Microphone désactivé : aucun son ne peut être envoyé.
    12. Arrêter ou autoriser le **son du microphone**.

        * Son arrêté : personne ne peut entendre l'utilisateur.
        * Son autorisé : les autres personnes peuvent entendre l'utilisateur.
    13. Activer ou désactiver les **invitations par SMS**.
 Les utilisateurs peuvent recevoir une invitation par message texte.
    14. Activer ou désactiver les **invitations par email**.
 Les utilisateurs peuvent recevoir une invitation par email.
3. Cliquez sur **Enregistrer**.

{% hint style="success" %}
La nouvelle configuration est automatiquement synchronisée.
Actualisez l'outil vidéo pour voir le nouveau workflow.
{% endhint %}
