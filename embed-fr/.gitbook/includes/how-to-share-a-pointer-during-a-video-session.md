# Comment partager un pointeur pendant une session vidéo

{% hint style="info" %}
Vous êtes maintenant dans une session vidéo.
{% endhint %}

1. Dans la barre d'outils, cliquez sur ![](../../.gitbook/assets/share-pointer-2025-03-05.png)pour activer un laser rouge.
2. Maintenez le clic gauche et déplacez le curseur pour tracer une ligne rouge sur le flux vidéo distant.

{% hint style="success" %}
*L'invité peut voir ce que vous dessinez avec le pointeur sur son écran.*
{% endhint %}
