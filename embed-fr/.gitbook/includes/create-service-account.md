1. Sélectionnez l'onglet **Informations**.
2. Cliquez sur **Créer** sous **Compte de service**.
3. Dans la fenêtre contextuelle **Nouveau compte de service** :

    1. Cliquez sur l'icône **Copier** à côté du **Client ID**.
    2. Enregistrez le **Client ID** dans votre intégration.
    3. Cliquez sur l'icône **Copier** à côté du **Client secret**.
    4. Enregistrez le **Client secret** dans votre intégration.

{% hint style="warning" %}
Conservez le Client secret en lieu sûr.
 Le Client secret n'est plus disponible après cette étape.
{% endhint %}
