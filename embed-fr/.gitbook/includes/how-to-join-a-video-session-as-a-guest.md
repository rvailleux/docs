# Comment rejoindre une session vidéo en tant qu'invité

{% hint style="info" %}
Vous avez reçu un message texte.
{% endhint %}

1. Dans le message, appuyez sur le lien.

    {% hint style="info" %}

    L'application invité s'ouvre dans le navigateur.

    {% endhint %}
2. Cochez les cases pour accepter les conditions légales.
3. Cliquez sur **Continuer**.
4. Autorisez le navigateur Web à utiliser le microphone et la caméra.
5. Vérifiez vos paramètres (caméra, microphone).
6. Cliquez sur **Rejoindre la session**.

{% hint style="success" %}
La page de session vidéo s'affiche.
{% endhint %}
