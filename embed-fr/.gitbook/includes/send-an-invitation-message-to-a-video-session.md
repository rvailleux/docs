# Envoyer un message d'invitation à une session vidéo

{% hint style="info" %}
Si Apizee Embed est intégré dans une plateforme, le numéro de téléphone de l'invité peut être automatiquement prérempli dans notre formulaire d'invitation.
Si le numéro de téléphone est correct, cliquez sur **Inviter et démarrer** pour lancer la session vidéo.
{% endhint %}

1. Pour ajouter un autre invité, saisissez un numéro de téléphone ou une adresse email dans le champ d'invitation.
2. Cliquez sur **Ajouter**. 

    {% hint style="warning" %}

    Vous ne pouvez inviter qu'un nombre maximum de participants autorisés à la session vidéo.

    {% endhint %}
3. Une fois tous les coordonnées saisies, cliquez sur **Inviter et démarrer**.

{% hint style="success" %}
La chronologie s'affiche. Elle indique le statut de chaque invité au fur et à mesure qu'ils rejoignent la session.
{% endhint %}
