1. Sélectionnez l'onglet **Session**.
2. Dans la liste, cliquez sur la session souhaitée.

    {% hint style="success" %}

    Les détails de la session s'affichent.

    {% endhint %}
3. Accédez à la section **Fichiers**.

# Télécharger un fichier

1. Survolez le fichier souhaité.
2. Cliquez sur le bouton de téléchargement.

{% hint style="success" %}
Le fichier est téléchargé sur votre appareil.
{% endhint %}

# Télécharger tous les fichiers

1. Cliquez sur **Tout télécharger**.
2. Attendez que le système génère le dossier compressé.

{% hint style="success" %}
Le dossier compressé contenant tous les fichiers est téléchargé sur votre appareil.
{% endhint %}
