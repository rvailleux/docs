{% hint style="info" %}
You want to create several accounts at the same time.
{% endhint %}

1. Open a spreadsheet document, such as Excel.
2. Fill in the table with:
    * First-name in column A
    * Last-name in column B
    * Username in column C
    * Password in column D
    * Email in column E
    * Nickname in column F 

        | ![Information](../../../.gitbook/assets/info.png) | Do no use spaces for the nicknames. 
Example: ~~docteur du jardin~~ &gt; **docteur\_du\_jardin** or **docteurdujardin** |
        | --- | --- |

![](../../../.gitbook/assets/EN-portal-create-user-spreadsheet.png)
3. **Save** the document in .csv format. 

![](../../../.gitbook/assets/EN-portal-create-user-save-csv-file.png)
4. On the Apizee portal, **choose** the document you saved earlier.
5. Choose the delimiter: **Semicolon (;)**
6. Click **Import**. 

![](../../../.gitbook/assets/EN-portal-create-user-import-csv-file.png)

{% hint style="success" %}
The accounts are created.
{% endhint %}
