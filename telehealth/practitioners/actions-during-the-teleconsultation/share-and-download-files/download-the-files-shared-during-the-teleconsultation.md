##  During the session

1. You received a notification in the **Messages**tab, click on the tab.
2. Click the **arrow** next to the image to download the file ![](../../../.gitbook/assets/button-download-apizee-meet.png) 
 
 ![](../../../.gitbook/assets/EN-conf-orga-download-file-in-messages-tab-click.png)

{% hint style="success" %}
The file is downloaded.
{% endhint %}

## After the session

{% hint style="info" %}
This end of session page with the shared files displays for the **patients** only.
 
If you are a **practitioner** and you want to download the files, you can get them in the **[Teleconsultation page](download-the-files-shared-during-the-teleconsultation.md)** in the portal.
{% endhint %}

{% hint style="info" %}
The session is over now.
The end of session page opens with the shared files on the right.
{% endhint %}

1. Click the **arrow** next to the file to download it. 
 **OR**
2. Click **Download all** to download all the files. 
 
 ![](../../../.gitbook/assets/EN-conf-end-of-session-page-shared-files-click.png)

{% hint style="success" %}
The files are downloaded.
{% endhint %}

If you want to, you can download those same files on another device. Interested? Click **Show more**.

 [-] [Hide](javascript:void%280%29)
 1. At the bottom right, click **Download on another device**. 
 
 ![](../../../.gitbook/assets/EN-conf-end-of-session-page-download-files-on-another-device-button.png)
2. Enter en email address and click **Confirm**. 
 
 ![](../../../.gitbook/assets/EN-conf-end-of-session-page-download-files-on-another-device-enter-address.png)

{% hint style="success" %}

3. From the other device, open the new message you have been sent on the email address given below.
4. Click the link in the message. 

{% hint style="success" %}
The end of session page opens with the shared files on the right.
{% endhint %}
## After the session in the portal

{% hint style="info" %}
You are the organizer of the session from which the files are from. Or, you are an administrator. 
 You are logged in to you account.
{% endhint %}

1. On the left-hand menu, click the service you want.
2. On the right, click **List**. 
That will help you find faster the session you are looking for.
3. In the list, find the session you want and click ![](../../../.gitbook/assets/button-edit.png) 
 
 ![](../../../.gitbook/assets/EN-portal-teleconsultation-edit-teleconsultation-page.png) 

{% hint style="success" %}
The page displays.
{% endhint %}
4. Under **Shared files**, select the file you want and click **Download**. 
 
 ![](../../../.gitbook/assets/EN-portal-conf-page-download-file-after-session.png)
