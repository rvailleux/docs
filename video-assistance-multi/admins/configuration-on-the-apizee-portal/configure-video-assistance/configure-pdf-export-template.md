# How to customize the PDF export?

{% hint style="info" %}
You want to customize what is displayed on the [PDF export](../../follow-up-the-assistances-on-the-portal/export-session-information-into-pdf.md) of a session.

{% endhint %}

1. In the left-hand menu, click the service you want.
2. Click the **Export PDF** tab.

![](../../../.gitbook/assets/EN-diag-export-pdf-template.png)
3. Tick the boxes in front of the information you want to be displayed.

    {% hint style="success" %}

    The **Preview** on the right automatically loads your changes.

    {% endhint %}
4. Click **Save**.

{% hint style="success" %}
The PDF template is customized and ready to be used.
{% endhint %}
