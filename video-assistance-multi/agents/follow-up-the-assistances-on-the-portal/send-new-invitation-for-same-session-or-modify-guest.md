# Need to reinvite, modify a guest or

# Change the invitation time?

1. In the left-hand menu, click the service you want.
2. In the ticket list, find the ticket you want and click ![](../../.gitbook/assets/button-edit.png)

![](../../.gitbook/assets/EN-portal-service-assistance-edit-ticket.png)
3. On the upper right side, click **Invite again**.
4. Choose another **date**and **time**, if needed, then click **Next**.
5. Change, add or delete the guest information, if needed.
6. Choose a message **template**and a **language**,****then click **Send**.

![](../../.gitbook/assets/EN-portal-assistance-multi-invite-again-button.png)

{% hint style="success" %}
All the guests receives a new invitation to the same session.

If you changed the guests information, it is updated and displays in the **Participants** section in the [assistance page](follow-a-ticket.md).
{% endhint %}
