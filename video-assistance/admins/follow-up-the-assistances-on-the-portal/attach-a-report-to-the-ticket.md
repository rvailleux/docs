#  Attach a report to the ticket

1. In the left-hand menu, click the service for which you want to attach a report.
2. In the ticket list, find the ticket you want to attach a report to and click ![](../../.gitbook/assets/button-edit.png) 
 
![](../../.gitbook/assets/EN-portal-service-assistance-edit-ticket.png)

    {% hint style="success" %}

    The ticket page displays.

    {% endhint %}
3. Go to the **Assignment** menu, in **Attach report**. 

    | ![](../../.gitbook/assets/tip.png) | If your company configured a report template, this message displays: **You can download a report template**
  -  Click **report template**. 
 
 <img src="../../.gitbook/assets/EN-portal-ticket-page-download-report-template.png" alt=""> 
 <img src="../../.gitbook/assets/ok.png"> <span>The report template downloads. 
 </span>  -  Open the document, fill it and save it.   |
    | --- | --- |
4. Click **Choose File** to import the report you just filled.
5. Click **Save**. 
 
 ![](../../.gitbook/assets/EN-Diag-portal-Attaching-report-to-ticket.png)

{% hint style="success" %}
The report is attached and display at the bottom of the **Ticket request** menu.
{% endhint %}

{% hint style="info" %}

*See also** [Configure a report template](https://doc.apizee.com/smart/project-diag-help-desk/configure-a-report-template)

{% endhint %}
