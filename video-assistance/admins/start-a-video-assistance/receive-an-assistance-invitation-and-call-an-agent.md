{% hint style="info" %}
You made a request for an assistance session and you received a message inviting you to participate to a remote assistance.
{% endhint %}

1. Open the invitation message for a remote assistance session. 
 
 ![](../../.gitbook/assets/EN_Diag-assistance_invitation_email_link.png)
2. Click the link.
3. Tick the boxes to agree to the terms then, click **Confirm**. 
 
 ![](../../.gitbook/assets/EN-Diag-consent-guest.png)
4. Click **Contact an agent**. 
 
 ![](../../.gitbook/assets/EN-Diag-contact_agent.png.png)
5. **Allow** the Web browser to use the microphone and the camera. 
 
 ![](../../.gitbook/assets/EN-Diag-contact-agent-allow-browser.png)

{% hint style="success" %}
The session starts.
{% endhint %}

![](../../.gitbook/assets/EN-Diag-assistance-session-starts.png)
