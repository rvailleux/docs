# Check the compatibility

Apizee solutions are "Plug-in free".
They are 100% Web-based to ensure communication experience on major Internet browsers and connected devices.

Video communication has never been that easy!

| No installation needed | The Apizee solutions use the **ApiRTC** architecture which uses the **WebRTC** protocol.<br> This protocol of communication is directly embedded in the latest versions of the Web browsers.<br> This is why you do not need to install anything as long as you have a compatible Web browser.<br> ![](../Storage/customer-engagement-user-en/project-content-reuse/schema_apirtc.png) |
