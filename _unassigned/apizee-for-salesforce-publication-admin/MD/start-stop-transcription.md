# How to start or stop the transcription

{% hint style="info" %}

The Admin has enabled transcription in the interactionTemplate configuration.<br><ul></ul>

{% endhint %}
**To start transcription:**
 
1. During a video call, click the **Transcription** button in the callbar.

{% hint style="success" %}

Transcription starts : A text panel appears at the bottom left of the screen. The panel shows each speaker's words in real time.

{% endhint %}
**To stop transcription:**

1. Click the **Transcription** button again.

{% hint style="success" %}

Transcription stops : The text panel disappears.

{% endhint %}
{% hint style="info" %}

When you start or stop transcription, the change applies to all participants in the call.<br>If the Admin configured transcription to start automatically, the text panel appears as soon as the call begins. You do not need to click the button to start it.

{% endhint %}
{% hint style="warning" %}

If the Transcription button does not appear in the callbar, transcription is not enabled in your configuration.<br>Contact your Admin.<br><ul></ul>

{% endhint %}
