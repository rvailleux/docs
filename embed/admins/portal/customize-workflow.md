1. Select the **Flow**tab.

    | ![](../../.gitbook/assets/ok.png) | A JSON file displays in a editor. 
This file contains all the configuration related to the video tool workflow. |
    | --- | --- |
2. Click into the editor to modify the followings:
    1. Show or hide the **on-screen pointer**.
The user can see and use the pointer on the screen.
    2. Show or hide **screen sharing**.
 The user can show their screen to other participants.
    3. Show or hide **remote photo capture**.
 The user can take a photo from a distance.
    4. Show or hide the **camera switch button**.
 The user can change the camera during the call.
    5. Show or hide **remote torch control**.
 The user can turn the phone light on or off.
    6. Show or hide the **support menu**.
 The user can access help.
    7. Choose the **default camera** for the main guest.

        * Front camera.
        * Back camera.
    8. Set the **maximum number of guests** allowed.
    9. Turn the **camera function** on or off.

        * Camera on: the image can be shown.
        * Camera off: no image is shown.
    10. Stop or allow the **camera image**.

        * Camera stopped: the image is not sent.
        * Camera allowed: the image is sent.
    11. Turn the **microphone function** on or off.

        * Microphone on: sound can be sent.
        * Microphone off: no sound can be sent.
    12. Stop or allow the **microphone sound**.

        * Sound stopped: nobody can hear the user.
        * Sound allowed: other people can hear the user.
    13. Turn **SMS invitations** on or off.
 Users can receive an invitation by text message.
    14. Turn **email invitations** on or off.
 Users can receive an invitation by email.
3. Click **Save**.

{% hint style="success" %}
The new configuration is automatically synchronized.
Refresh the video tool to see the new workflow.
{% endhint %}
