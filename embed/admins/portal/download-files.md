1. Select the **Session** tab.
2. In the list, click the session you want.

    {% hint style="success" %}

    The session details display.

    {% endhint %}
3. Go to the **Files** section.

# Download one file

1. Hover the file you want.
2. Click download button.

{% hint style="success" %}
The file is downloaded on your device.
{% endhint %}

# Download all the files

1. Click **Download all**.
2. Wait for the system to generate the compressed folder.

{% hint style="success" %}
The compressed folder with all the files is downloaded on your device.
{% endhint %}
