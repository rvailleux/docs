1. In Genesys Admin, go to **Integrations**.
2. Delete the Apizee widget.
3. In Admin, go to **Groups** and delete the group **APIZEE\_WIDGET\_Agents**.
4. In Admin, go to **Roles / Permissions** and delete the **APIZEE\_WIDGET\_Role** role.

{% hint style="success" %}
*The widget has been successfully uninstalled.*
 
The system is now clean and ready for reinstallation or removal.
{% endhint %}
